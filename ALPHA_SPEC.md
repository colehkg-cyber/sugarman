# 🍬 슈가맨 — Alpha Spec v0.1

> 입자 큰 산설탕이 그릇에 눈처럼 쌓이듯, 달콤한 만남이 내려온다.

---

## 1. 서비스 한 줄 요약

경제력 있는 남성과 매력적인 여성을 연결하는 프리미엄 매칭 웹서비스.
**남성이 대화를 걸 때 과금**하는 메시지 기반 수익 모델.

---

## 2. 유저 타입

### 👔 슈가 (Sugar) — 남성
- 30대 후반~50대, 경제력 보유
- 10~20살 어린 여성과의 만남 목적
- **돈을 쓰는 쪽** (과금 주체)

### 🍭 캔디 (Candy) — 여성  
- 20대~30대 초반
- 프로필 등록 후 매칭 대기
- **무료 사용** (공급 확보가 핵심)

---

## 3. 알파 핵심 기능 (MVP)

### 3.1 가입/인증
| 기능 | 슈가(남) | 캔디(여) |
|------|----------|----------|
| 가입 | 휴대폰 인증 | 휴대폰 인증 |
| 본인확인 | 신분증 인증 (성인) | 신분증 인증 (성인) |
| 프로필 | 직업/연수입 레벨 선택 | 사진 + 자기소개 |
| 심사 | 없음 (알파) | 사진 심사 (수동, 알파) |

### 3.2 탐색 (Discovery)
- **피드 형태**: 캔디 프로필 카드 리스트 (사진 + 한 줄 소개 + 나이 + 지역)
- **필터**: 나이, 지역, 체형
- **슈가만 탐색 가능** — 캔디는 받은 메시지함만 확인
- 비대칭 구조: 슈가가 고르고, 캔디가 수락/거절

### 3.3 메시지 (핵심 과금 포인트)
```
슈가가 캔디에게 첫 메시지 전송 → 슈가 코인 차감
                                  ↓
              캔디가 수락하면 → 대화방 오픈 (24시간)
              캔디가 거절하면 → 코인 반환 없음
                                  ↓
              24시간 이후 연장 → 추가 코인 차감
```

- **첫 메시지**: 3코인 (약 3,000원)
- **대화방 연장 (24h)**: 1코인 (약 1,000원)  
- **연락처 교환 요청**: 10코인 (약 10,000원)
- 캔디는 무료

### 3.4 코인 시스템
| 패키지 | 코인 | 가격 | 단가 |
|--------|------|------|------|
| 체험 | 5코인 | 5,000원 | 1,000원 |
| 기본 | 30코인 | 25,000원 | 833원 |
| 프리미엄 | 100코인 | 70,000원 | 700원 |
| VIP | 300코인 | 180,000원 | 600원 |

### 3.5 프로필 상세
**슈가 프로필:**
- 닉네임 (실명 비공개)
- 연령대
- 직업 카테고리 (사업가/전문직/임원/기타)
- 연수입 레벨 (5천만~1억 / 1억~3억 / 3억~5억 / 5억+)
- 자기소개 (200자)
- 사진 (선택, 블러 가능)

**캔디 프로필:**
- 닉네임
- 나이, 키, 체형
- 직업/학교 (선택)
- 사진 3~6장 (필수, 심사 통과)
- 자기소개 (200자)
- 원하는 만남 스타일 태그 (#식사 #카페 #드라이브 #여행 #쇼핑)

## 6. 화면 구성 (알파 필수)

```
[공통]
├── 랜딩 페이지 (서비스 소개 + 가입 유도)
├── 회원가입 (성별 선택 → 분기)
├── 로그인
└── 마이페이지 (프로필 수정 + 코인 잔액)

[슈가 전용]
├── 탐색 피드 (캔디 리스트)
├── 캔디 프로필 상세
├── 메시지 보내기 (코인 차감)
├── 대화함 (진행 중 대화 목록)
└── 코인 충전

[캔디 전용]
├── 받은 메시지함 (수락/거절)
├── 대화함 (진행 중 대화 목록)
└── 프로필 관리

[관리자]
└── 캔디 사진 심사 (수동)
```

## 7. 데이터 모델 (핵심)

```
User
├── id, type(SUGAR|CANDY), phone, nickname
├── age, gender, region
├── profile (JSON: 직업, 수입레벨, 사진URLs, 소개, 태그 등)
├── coin_balance (슈가만)
├── verified (본인인증 여부)
└── status (ACTIVE|SUSPENDED|PENDING_REVIEW)

Conversation
├── id, sugar_id, candy_id
├── status (PENDING|ACTIVE|EXPIRED|REJECTED)
├── expires_at
└── created_at

Message
├── id, conversation_id, sender_id
├── content (text)
└── created_at

CoinTransaction
├── id, user_id, amount(+/-), type
├── (PURCHASE|MESSAGE|EXTEND|CONTACT_REQUEST)
└── created_at

PhotoReview (관리자)
├── id, user_id, photo_url
├── status (PENDING|APPROVED|REJECTED)
└── reviewed_at
```

## 8. 법적 고려사항 ⚠️

> 알파 전 반드시 법률 검토 필요

1. **성매매알선 등 행위의 처벌에 관한 법률** — "만남 주선"이 알선으로 해석될 리스크
2. **청소년성보호법** — 미성년자 완전 차단 필수 (본인인증 강제)
3. **개인정보보호법** — 사진/신분증 처리, 동의 절차
4. **전자금융거래법** — 코인 시스템이 선불전자지급수단에 해당 가능
5. **통신사업 신고** — 부가통신사업자 신고 필요 가능
6. **이용약관** — "데이트 매칭" 명목, 성매매 금지 조항 명시

**대응 방향:**
- 이용약관에 "성적 서비스 거래 목적 이용 금지" 명시
- 신고 시스템 + 운영 정책으로 리스크 관리
- 법률 자문 받고 시작 (알파 전)
